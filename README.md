## Требования для системы

Terraform 1.2.9

Ansible 2.13.2

GNU Make 4.2.1

## Инструкции для приложения

Инициализация terraform
```shell
make init
```

Подготовка инфраструктуры
```shell
make apply
```

Деплой приложения
```shell
make deploy
```
